# 💳 Credit Portfolio Analytics

## 🚀 Overview

This project focuses on analyzing credit portfolios to assess risk, forecast defaults, and optimize lending strategies. Using advanced analytics, statistical modeling, and visualization, the repository demonstrates how financial institutions can make data-driven decisions while minimizing risk exposure.  

---

## 🎯 Objectives

* Analyze credit portfolio performance and risk metrics  
* Identify high-risk segments and customer profiles  
* Build predictive models for default probability  
* Visualize portfolio trends and insights for decision-making  

---

## 🛠 Tech Stack

**Programming & Analytics:**  
* Python (Pandas, NumPy, Scikit-learn)  
* SQL (PostgreSQL, MySQL)  
* R (Regression, Logistic Modeling)  

**Big Data & Processing:**  
* PySpark  
* Hadoop HDFS  

**Cloud & Storage:**  
* AWS S3  
* AWS RDS  

**Visualization:**  
* Tableau  
* Plotly / Dash  

---

## 🔄 Data Engineering Pipeline

* Collected and cleaned datasets from banking and financial sources  
* Built ETL pipelines to integrate data from RDS, CSVs, and APIs  
* Standardized and validated data for model readiness  
* Processed large datasets efficiently using PySpark  

---

## 📊 Risk Modeling & Analytics

* Conducted credit scoring and risk segmentation  
* Built Logistic Regression, Random Forest, and XGBoost models to predict defaults  
* Performed feature engineering and correlation analysis  
* Validated models using precision, recall, and ROC-AUC metrics  

---

## 📈 Business Impact

* Identified high-risk borrowers for proactive management  
* Optimized credit portfolio performance  
* Reduced default rates through predictive insights  
* Supported strategic lending decisions
